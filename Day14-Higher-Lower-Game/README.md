The **Higher Lower Game** is a command-line game where the player compares two public figures or brands and guesses which one has more followers.

## ğŸ§  Game Concept

- Two accounts are displayed: **Account A** and **Account B**
- Each account has:
  - Name
  - Description
  - Country
- The player guesses who has more followers by typing **A** or **B**
- If the guess is correct:
  - The score increases
  - Account B becomes the next Account A
- If the guess is wrong:
  - The game ends
  - Final score is displayed

## ğŸ® How to Play

1. Run the `main.py` file

   - `B` if you think Account B has more followers
4. Keep playing until you guess wrong
